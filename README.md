License : GPLv2
License URI : http://www.gnu.org/licenses/gpl-2.0.html

## Description
Hightlight your blocks and stop clicking everywhere to find them.


## Installation
1. Install the plugin and activate.
2. Use shortcut : Ctrl + Alt + R
